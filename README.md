# 🌾 OptiCrop: Smart Agricultural Production Optimization Engine

OptiCrop is an AI-powered agricultural recommendation system designed to optimize regional crop production by integrating multi-parametric soil metrics and environmental factors across three unique execution scenarios.

---

##QUICK START / ACTIVATION CODE

Whenever your Codespace restarts or deactivates, copy and paste this **single line** into your terminal to instantly launch the application:
bash
```
cd "5. Project Development Phase/code" && python app.py
```

---

##Project Architecture Layout

The application utilizes a single-folder code architecture to eliminate directory fragmentation:

```text
.
├── 5. Project Development Phase/
│   └── code/
│       ├── app.py              # Main Application Server (Flask)
│       ├── train.py            # Model Training Script
│       ├── data/               # Crop Recommendation Dataset
│       ├── models/             # Serialized ML Model Artifact (.pkl)
│       └── templates/          # Streamlit-Style UI Templates
├── 6. Project Testing/         # Automated Unit Tests
├── 7. Project Documentation/   # System Architecture Guidelines
├── 8. Project Demonstration/   # Presentation Outlines
└── README.md                   # Workspace Manifest & Activation Guide
```
